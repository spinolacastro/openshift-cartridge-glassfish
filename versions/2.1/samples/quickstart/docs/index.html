The QuickStart Sample Application

The QuickStart sample is a small web application that was extracted from the Java Enterprise Edition (Java EE) tutorial for use in the Quick Start Guide to the Sun Java System Application Server 9.0 located at <install_dir>/docs/QuickStart.html. This sample is a customized "Hello World" application which is implemented with a pair of dynamic JavaServer Pages (JSP).
