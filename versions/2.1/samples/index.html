Sun Java System Application Server 

Application Server - Sample Applications

The Application Server quickstart has one sample application, GlassFish with Clustering and Loadbalancing, which is useful primarily for confirming successful installation and operation of Application Server with Clustering features.

More extensive sample applications are available for download in the following packages:

  • Java Enterprise Edition 5 Tutorial
  • Java EE 5 SDK Samples bundle
  • J2EE 1.4 SDK Samples bundle

Installing Other Samples Packages

The downloadable bundles for the Java EE Tutorial or the Java EE 5 and J2EE 1.4 samples contain specific instructions for installing and working with the samples.

After downloading, install Java EE 5 samples under samples directory and access the javaee5 samples documentation for instructions on building and running the samples.

Similarly , after downloading, install J2EE 1.4  samples under samples directory and access the j2ee14 samples documentation for instructions on building and running the samples.

Before working with the samples, follow the Quick Start Guide to familiarize yourself with the features and basic operation of Application Server. In addition, review the following section.

Setting Up Your Environment

After following the instructions in the Quick Start Guide to ensure that Application Server is installed and functioning properly, add the following directory to your PATH environment variable: install_dir/bin. Doing this enables you to run the asant build tool which is needed for Quickstart and J2EE 1.4 samples.
